# Flutter Development

## VS Code

We use `Visual Studio Code` for general development

## User setttings

There are the required VS Code settings. Set globally in user settings (recommended), or per project.

```json
{
    "telemetry.telemetryLevel": "off",
    "files.trimFinalNewlines": false,
    "editor.rulers": [
        120
    ],
    "cSpell.diagnosticLevel": "Hint"
}
```
